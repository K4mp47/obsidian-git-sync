
    ARM64vm   v1.0  2022
===================================================

Macchina virtuale ARM Cortex A53 basata su
Linux Debian creata per il corso di Architettura
degli Elaboratori mod. 2


Per eseguire:
1) Installare QEmu (Versione 6.1 o superiore)
2) Eseguire run_vm.bat in ambiente Windows o run_vm.sh in OSX/Linux
3) Attendere il boot
4) Effettuare il login nella VM via ssh:

$ ssh vm@localhost -p 2222

password: vm
